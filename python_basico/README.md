# Práctica de Python básico

En esta práctica se trabajan los conceptos fundamentales de Python:
sentencias básicas, bloques lógicos y estructuras de control.

## Contenido
- Variables y tipos de datos
- Condicionales (if, elif, else)
- Bucles (for, while)
- List comprehension

## Entorno
- Jupyter Notebook
- Visual Studio Code

## Objetivo
Afianzar las bases de Python mediante ejercicios prácticos y progresivos.
