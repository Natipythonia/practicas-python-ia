# Análisis Exploratorio de Datos (EDA)

Proyecto de análisis exploratorio de datos realizado como parte del
Máster en Inteligencia Artificial y Big Data.

## Contenido
- Carga y exploración inicial del dataset
- Limpieza y preparación de datos
- Análisis estadístico descriptivo
- Visualización de datos

## Tecnologías
- Python
- Jupyter Notebook
- Pandas
- Matplotlib / Seaborn

## Objetivo
Analizar un conjunto de datos para extraer conclusiones relevantes
y detectar patrones mediante técnicas de EDA.
