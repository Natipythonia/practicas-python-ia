# Prácticas de Python – Máster en Inteligencia Artificial

Este repositorio contiene prácticas realizadas en Python como parte del
Máster en Inteligencia Artificial y Big Data.

Los ejercicios han sido desarrollados en **Jupyter Notebook** y
**Visual Studio Code**, siguiendo un enfoque didáctico y progresivo.

## Tecnologías utilizadas
- Python
- Jupyter Notebook
- Visual Studio Code
- Linux

## Contenido
- Sentencias básicas y bloques lógicos
- Funciones y ámbito (scope)
- Programación orientada a objetos
- Uso de módulos y librerías
- Análisis exploratorio de datos (EDA)

## Objetivo
Mostrar competencias prácticas en Python y análisis de datos,
con código claro, comentado y orientado a buenas prácticas.
